# CLAUDE.md — Grocery Planner

---

## Project Goal

A weekly grocery deal tracker for Vancouver. Fetches flyer data from the Flipp API for 6 stores (Real Canadian Superstore, Shoppers Drug Mart, No Frills, FreshCo, Walmart, Costco), classifies ~2300 items using a local LLM (Ollama qwen2.5:14b), and generates a mobile-friendly HTML report showing deals on weekly staples and tiered food categories. Report is deployed to GitHub Pages and checked on phone while grocery shopping.

- Run weekly on Thursdays when new flyers drop
- Staples: items bought every week — find cheapest price across all stores
- Tier lists: food items ranked T1–T4 by nutrition quality (meat, carbs, vegetables, fruit)
- Vegetables for Meals: derived from tier results, filtered to meal-sized veg only
- LLM does classification; avoid keywords in config filter false positives
- Deployed automatically to GitHub Pages on each full run

## File Locations

```
grocery_planner/
├── todo.md                  <- Active tasks only
├── log.md                   <- Completed work history (newest first)
├── run.py                   <- Entry point: python run.py [--report-only] [--dump]
├── index.html               <- Generated report (deployed to GitHub Pages)

│   ├── tier_lists.yaml      <- Ranked food items (meat/carbs/veg/fruit) with avoid keywords
│   ├── preferences.yaml     <- Weekly staples + meal_vegetables whitelist
│   └── settings.yaml        <- Ollama URL, model, postal code
```
